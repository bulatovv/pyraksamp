# Recipes

Short, self-contained scripts for common tasks.

| [Key Presser](key-presser.md) | Timed key presses and vehicle control |
| [PIN Pad](pin-pad.md) | Click on-screen textdraw number buttons to enter a PIN |
| [Reconnect](reconnect.md) | Auto-reconnect with retry and exponential backoff |
| [Multi-Bot](multi-bot.md) | Run multiple bots concurrently, with proxy rotation |
